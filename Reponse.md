 Quelle est la différence entre Git et GitHub ?
Git est un système de gestion de versions qui permet de suivre les modifications d’un projet.
GitHub est une plateforme en ligne qui permet d’héberger et partager les projets Git.
À quoi sert la commande git init ?
La commande git init sert à initialiser un dépôt Git dans un dossier.
Quelle commande permet d’envoyer un projet sur GitHub ?
git push origin main
Pourquoi utilise-t-on les branches ?
Les branches permettent de travailler sur de nouvelles fonctionnalités sans modifier directement le projet principal.
Quelle commande permet de voir l’historique des commits ?
git log
